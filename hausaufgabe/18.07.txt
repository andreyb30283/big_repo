Давайте разберемся с этой задачей по шагам:

Создание папки с именем на сервере:
Для создания папки с вашим именем на сервере linux.itcareerhub.de, используйте команду mkdir. Например:
mkdir YourName
Замените YourName на ваше реальное имя.
Создание скрипта:
Внутри папки с вашим именем создайте файл task_NAME.sh, где NAME — ваше имя.
Откройте файл для редактирования (например, с помощью nano):
nano task_NAME.sh

Вставьте следующий код в файл:
#!/bin/bash
for i in {1..10}; do
    touch "${i}_$(date +'%d.%m.%y')"
done
Этот скрипт создаст 10 файлов с порядковыми номерами и текущей датой в формате “1_20.04.23”, “2_20.04.23” и так далее.
Создание задания в планировщике:
Откройте планировщик задач (crontab) для редактирования:
crontab -e

Добавьте следующую строку, чтобы запускать ваш скрипт каждые 15 минут:
*/15 * * * * /путь_к_папке_с_вашим_именем/task_NAME.sh
Замените /путь_к_папке_с_вашим_именем на реальный путь к папке с вашим именем.

Создать скрипт, который создаст 10 файлов со случайными числами, а затем заархивирует директорию с этими файлами.

#!/bin/bash 
# Создаем временную директорию 
mkdir-p /home/tar temp_dir="/home/tar" 
# Создадим файлы for i in {1..10};
do echo "Случайное число: $RANDOM" > "$temp_dir/$i.txt"; 
done 
# Создаем архив из временной директории 
tar-czf /tmp/RANDOM.tar.gz-C $temp_dir . 
# Удаляем временную директорию 
rm-r $temp_dir
